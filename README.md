# image-maker
  * Pythonで書かれた画像作成用のCLIコマンド

## ●使い方
```
Usage: python image-maker.py script-file
```
  * 指定した`script-file`を読み込み、1行ずつ実行する

## ●スクリプト記法

### コメント
```
# 任意のテキスト
```
  - 文字 `#` で始まる行はコメントであり、読み飛ばされる

### 代入と計算
```
変数名 = 式
```
  - 変数に式の値（数値, 文字列, PIL.Image形式の画像）を格納する
  -	変数名は、Python の変数名として有効な名前のみ使える

  +	右辺の式で、Python の演算子が使える（`1 + 2 / 3` など)
  + 式で Python のドット記法が使える（`img.width` など）
  + 式で Python の f-string が使える（`f"{result:6.2f}"` など）
  + 式で いくつかの組み込み関数が使える（`int(100 / 3)` など）

右辺の式で使える組み込み関数

|関数|説明|
|:--|:--|
|`int(式)`|式の値を整数に変換する（小数点以下切り捨て）|
|`max(式1,式2)`|大きいほうの値を返す|
|`min(式1,式2)`|小さいほうの値を返す|
  * バージョンupで追加するかもしれない

### メッセージ出力
```
echo 引数1 引数2 ...
```
  -	指定した引数を空白区切りで出力する
  -	引数は（変数, 数値, 文字列）が指定可能で、式は指定できない
  -	出力フォーマットを指定したい場合は、代入文で変数に格納して出力する<br>
	```
	msg = f"答={result:6.2f}"
	echo msg
	```

### コマンド呼び出し
```
コマンド名 引数1 ...
```
  - 各種コマンドを実行する
  - コマンド名は任意の文字列なので、名前に`-`などが入る場合もある
  - 引数は (変数名, &変数名, 数値, 文字列) が指定可能で、式は指定できない
  - "&変数名" は参照渡しで、指定した変数をコマンド側で書き換えることが可能
