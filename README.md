# image-maker
  * Pythonで書かれた画像作成用のCLIコマンド

## ●使い方
```
Usage: python image-maker.py script-file
```
  * 指定した`script-file`を読み込み、1行ずつ実行する

## ●スクリプト記法

### コメント
```
# 任意のテキスト
```

  * 文字 `#` で始まる行はコメントであり、読み飛ばされる

### 代入文
```
変数名 = 式
```
  * 変数に、数値 / 文字列 / 画像 (PIL.Image形式）を格納する
  *	変数名は、Python の変数名として有効な名前のみ使える
  *	右辺の式では、Python の演算子が使える
  * 右辺の式では、int() などいくつかの組み込み関数が使える
  * 右辺の式では、Python の f-string が使える

### 代入文の右辺で使える組み込み関数
|関数|説明|
|:--|:--|
|`int(式)`|式の値を整数に変換する（小数点以下切り捨て）|
|`max(式1,式2)`|大きいほうの値を返す|
|`min(式1,式2)`|小さいほうの値を返す|

### echo文
```
echo 引数1, ...
```
  *	指定した引数を空白区切りで出力する
  *	引数は（変数、数値、文字列）が指定可能で、式は指定できない
  *	出力フォーマットは指定できない
  *	代入文で f-string を利用し値をフォーマットして表示することは可能
